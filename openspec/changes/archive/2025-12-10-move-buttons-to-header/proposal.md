# Change: Move Send and Clear buttons to header

## Why
Move the Send and Clear shortcut buttons from the bottom button group to the header area to improve accessibility and reduce user interaction overhead.

## What Changes
- Move Send and Clear buttons from button-group below input to header (right side)
- Keep existing functionality (keyboard shortcuts, gestures)
- Maintain responsive design for landscape mode
- Update mobile UI layout to accommodate new header structure

## Impact
- Affected specs: mobile-ui
- Affected code: src/templates/index.html
- No breaking changes to existing functionality
